#Security Toolkit 

Proyecto educativo en Python que simula herramientas básicas de ciberseguridad.

## Características
- Información del sistema
- Escáner educativo de vulnerabilidades
- Interfaz por consola con colores
- Efectos visuales (typewriter)

## Estructura
- `monolitico/` versión en un solo archivo
- `modular/` versión con programación modular

## Tecnologías
- Python 3
- Librerías estándar (`os`, `platform`, `shutil`, etc.)

## Aviso
Este proyecto **NO realiza ataques reales**.
Está diseñado únicamente con fines educativos.
